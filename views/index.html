Timestamp Microservice

Endpoints disponibles

Método Ruta Descripción
GET /api Devuelve la fecha y hora actuales en formato Unix y UTC.
GET /api/2015-12-25 Devuelve la fecha enviada en formato ISO.
GET /api/1451001600000 Devuelve la fecha correspondiente al timestamp Unix enviado.
POST /api/secure Devuelve una fecha a partir de un JSON con año, mes, día y opcionalmente hora, minuto y segundo.

Ejemplo de cuerpo JSON para POST

{
  "año": 2025,
  "mes": 5,
  "día": 25,
  "hora": 14,
  "minuto": 30,
  "segundo": 0
}

Nota: Los campos hora, minuto y segundo son opcionales.

Ejemplo de salida

Formato Valor
unix 1451001600000
utc Fri, 25 Dec 2015 00:00:00 GMT
